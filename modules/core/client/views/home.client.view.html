Welcome

GEKO (GUI for the Evaluation of Knee Osteoarthritis) is a custom image grading system that evaluates frontal histological images of rodent knee OA. GEKO reads in histological images and asks the user to identify certain features within the joint before calculating the OARSI scores, saving the marked up images, and writing the information to a result page.
